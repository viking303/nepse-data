# NEPSE Daily Price Data

Auto-updated every weekday at 3:15 PM NPT using GitHub Actions.

## Data
- `daily_price_data/` — daily CSV files with stock prices from merolagani.com

## Columns
| Column | Description |
|---|---|
| Date | Scrape date (NPT) |
| # | Serial number |
| Symbol | Stock symbol |
| LTP | Last traded price |
| % Change | Change from previous close |
| High | Day's high price |
| Low | Day's low price |
| Open | Opening price |
| Qty. | Quantity traded |
| Turnover | Total turnover |
| Page | Page number from source |
